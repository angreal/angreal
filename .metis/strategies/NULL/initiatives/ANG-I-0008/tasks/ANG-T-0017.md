### Quick Start
```python
from angreal.integrations.flox import Flox, flox_required

# Decorator pattern
@flox_required(".")
def my_task():
    pass

# Context manager
with Flox(".") as flox:
    pass

# Explicit control
flox = Flox(".")
flox.activate()
```
